croc — это инструмент для простой и безопасной передачи файлов между любыми двумя компьютерами.

<b>Основные возможности</b>
Поддержка передачи данных между <b>любыми двумя компьютерами</b> (через ретранслятор)
<b>Сквозное шифрование</b> (на базе протокола PAKE)
<b>Кросс-платформенная работа</b> (Windows, Linux, macOS)
Передача <b>нескольких файлов</b> одновременно
Не требует настройки <b>локального сервера</b> или проброса портов
Приоритетная работа через <b>IPv6</b> с автоматическим переходом на IPv4

Командная версия приложения, на которой основан этот интерфейс, доступна здесь:

https://github.com/schollz/croc
